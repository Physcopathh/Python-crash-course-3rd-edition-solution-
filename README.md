## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/Physcopathh/Python-crash-course-3rd-edition-solution-.git
```

2. Navigate into the folder:
```bash
cd Python-crash-course-3rd-edition-solution-
```

3. Run Python scripts:
```bash
python Chapter_1/hello_world.py
```
